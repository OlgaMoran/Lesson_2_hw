## Списки

Чтобы создать ненумернованный список, необходимо пункты выделить звездочкой (*).
Например, вот так:

* элемент 1
* элемент 2
* элемент 3

Чтобы создать нумерованный список, необходимо пункты просто пронумеровать. Например, вот так:

1. Пункт 1
2. Пункт 2
3. Пункт 3
